# TASKORA

A full-stack task management application built with React, Django REST Framework, and JWT Authentication.

## Features

- User Registration & Login
- JWT Authentication
- Create Tasks
- Edit Tasks
- Delete Tasks
- User Profile
- Task Dashboard
- Responsive Design

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS
- Axios
- React Router

### Backend
- Django
- Django REST Framework
- Simple JWT
- SQLite

### Frontend

cd Taskflow-Frontend
